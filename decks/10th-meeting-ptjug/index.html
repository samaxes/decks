Adopt a JSR

What is the Adopt a JSR program?

  • A JUG-lead global initiative to improve Java standards
  • Groups of JUG members work on JSRs
  • See http://adoptajsr.org for full details

Why should the JUGs get involved?

  • To move the entire Java ecosystem forward
  • To be at the forefront of new technology
  • To boost the careers of JUG members
  • To acquire new knowledge
  • To gain competitive advantage
  • It's fun!

How can you help?

  • Help moderate the mailing lists police towards helpful conversations
  • Help evangelise the JSR social networks, blogging, talks
  • Help with documentation translate into PT
  • Improve project infrastructure and JSR visibility hosting (java.net, GitHub), SEO of website

How else can you help?

  • Test the early RI builds report bugs, suggest enhancements
  • Help triage issues reproduce issues, erase/merge duplicates, set priorities
  • Give feedback on design how you would use the JSR as a developer
  • Help build the RI implement the spec
  • Help build the TCK fix the test suite
